# Buttons

Buttons will navigate to either other categories or open files. You can set different options in **_ui_settings.csv** with key "layout" or "display". The defaut settings are 'list' for "layout" and "none" for "display".

## Available Layouts

|Layout        |Description                           |
|--------------|--------------------------------------|
|list          |menu-like list                        |
|tiles         |grid layout with two buttons in a row |

## Available Displays

|Layout          |Description                            | 
|----------------|---------------------------------------|
|none            |nothing                                |
|image           |small image next to the title          |
|icon (inactive) |small one-colored icons                |
