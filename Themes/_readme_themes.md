# Themes

The theme defines the colors of your section. If you set the theme on one category, all category and files below will have the same theme. The default theme is "medieval".

Add the theme to the **_ui_setting.csv** with key "theme". 

## Example

theme,medieval

## Available Themes
| UI Setting    | Colors                 |
|---------------|-----------------------------|
| breton        | blue, white                 |
| darkFantasy   | dark brown, black           |
| japanese      | light pink                  |
| medieval      | light brown, brown          |
